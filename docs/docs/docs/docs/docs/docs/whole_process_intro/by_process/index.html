根据过程分

根据过程分

先按照汽车销售的先后流程，基本上可以分成两个大的阶段：

  • 售前：
    • 收集线索
    • 过滤线索，得到有效线索，跟进了解客户详情后，建卡
    • 对于有望客户进行跟进
    • 直到下单变订单客户
    • 对订单客户跟进，直到成交
  • 售后
    • 对成交后提了车的客户进行回访提高满意度
    • 在订单和交车期间都会涉及到很多相关内容，称为边际，包括保险、信贷、装潢美容等
